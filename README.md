# midi-monitor

A browser-based tool to monitor incoming midi (mpe & regular) messages from any midi port or device connected. 

Live at [tools.mattdonald.com](https://tools.mattdonald.com)

---

## files

**midi-monitor-readymag.html**
The version embedded in the Readymag site via code widget. Uses Readymag's internal font names for [tools.mattdonald.com](https://tools.mattdonald.com) `custom_175773` & `custom_175809` to inherit the site's typography.
